# Reducing LLM Hallucinations with Knowledge Graphs: A Survey

This repository contains the code, data, and intermediate results used to reproduce the literature search and analysis conducted for surveying the literature on **reducing hallucinations in LLMs with knowledge graphs**.

The repository provides the workflow for retrieving, processing, merging, and analyzing the literature considered in the survey.

## Repository Structure

* **`data/`** – Raw data used in the literature review, including exports from ACM Digital Library, IEEE Xplore, SpringerLink, and a preprocessed ACL Anthology dataset.
* **`src/`** – Python notebooks implementing the literature review methodology, including literature retrieval, preprocessing, and analysis.
* **`results/`** – Intermediate search results generated by the literature retrieval pipeline.
* **`files/`** – Final merged literature records, deduplicated records, and figures produced during the analysis.

```text
.
├── data/               # Raw literature search exports and ACL Anthology dataset
├── files/              # Processed datasets, deduplicated results, and figures
├── results/            # Intermediate literature retrieval outputs
├── src/                # Notebooks for retrieval, preprocessing, and analysis
│   ├── acl_parsing.ipynb
│   ├── literature_review.ipynb
│   └── analysis.ipynb
├── requirements.txt    # Python dependencies
├── .env                # API keys (user-created, not tracked)
├── .venv/              # Python virtual environment (user-created)
└── README.md
```

## Reproducing the Literature Search

1. Clone the repository.

2. Create and activate a Python virtual environment (e.g., `.venv`).

3. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Create the `.env` file and add the API keys required to run `src/literature_review.ipynb`, e.g., SCOPUS_API_KEY.

5. Execute the notebooks in the `src/` directory to reproduce the literature retrieval and analysis pipeline.
